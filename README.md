## Проектная работа 8
"Рефакторинг и сборка Вебпаком"

Сервис Mesto - это интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

**Использованные технологии:**
1. HTML
2. CSS
3. JavaScript
4. NPM
5. Webpack

**Реализация:**
* Grid Layout
* Flexbox
* Медиазапросы
* Nested БЭМ
* ОПП
